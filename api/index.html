# Endpoints

Semua endpoint hanya mendukung method GET. Klik endpoint untuk mencobanya langsung.

GET /data/projects.json List semua project ▼

Mengembalikan array berisi data project portofolio. Cocok untuk latihan menampilkan card list dari JSON.

GET /data/skills.json List skills & tech stack ▼

Mengembalikan array berisi data skills dan tech stack. Cocok untuk latihan render tag/badge dari data JSON.

GET /data/profile.json Data profile pemilik ▼

Mengembalikan object berisi data profile. Cocok untuk latihan menampilkan data single object dari JSON.

⚠️ API ini hanya mendukung method GET. Untuk latihan POST, PUT, DELETE — jalankan Express server di local. Tombol "Coba Sekarang" akan aktif setelah file JSON kamu upload ke Vercel.

Fetch data dengan async/await

JavaScript 
async function ambilProjects() {
  try {
    const res  = await fetch('https://agungwin22.vercel.app/data/projects.json');

    if (!res.ok) throw new Error(`HTTP error: ${res.status}`);

    const data = await res.json();
    console.log(data);

  } catch (error) {
    console.log('Gagal:', error.message);
  }
}

ambilProjects();

Fetch beberapa endpoint sekaligus dengan Promise.all

JavaScript 
async function ambilSemua() {
  const baseUrl = 'https://agungwin22.vercel.app';

  const [projects, skills, profile] = await Promise.all([
    fetch(`${baseUrl}/data/projects.json`).then(r => r.json()),
    fetch(`${baseUrl}/data/skills.json`).then(r => r.json()),
    fetch(`${baseUrl}/data/profile.json`).then(r => r.json()),
  ]);

  console.log(projects, skills, profile);
}

ambilSemua();

Tampilkan data ke HTML

JavaScript 
async function tampilProjects() {
  const res      = await fetch('/data/projects.json');
  const projects = await res.json();

  const container = document.getElementById('projects');

  projects.forEach(project => {
    container.innerHTML += `
      <div class="card">
        <h3>${project.nama}</h3>
        <p>${project.deskripsi}</p>
      </div>
    `;
  });
}

tampilProjects();
